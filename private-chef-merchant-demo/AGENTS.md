# private-chef-merchant-demo 规则

本目录用于“私厨上门商户端”手机端纯前端静态演示。

## 范围

- 只使用 HTML、CSS、原生 JavaScript。
- 不接后端、不接数据库、不接真实接口。
- 不做真实登录、鉴权、支付、结算或文件上传。
- 页面数据来自静态 HTML 或 `assets/js/mock.js` 的本地演示交互。
- 页面跳转使用普通 `a` 标签 `href`。

## 页面定位

本目录面向入驻店铺、私厨团队和商家负责人，用于演示店铺、订单、厨师、套餐、菜品、财务和经营数据管理流程。

不要把文案写成普通外卖用户端，也不要写成厨师个人接单端。
