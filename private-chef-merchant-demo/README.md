# 私厨上门商户端静态演示

本目录是“私厨上门商户端”手机端纯前端静态演示页面，不包含后端、数据库、真实接口、真实登录鉴权或真实上传能力。

## 演示入口

直接在浏览器打开：

```text
private-chef-merchant-demo/index.html
```

## 推荐流程

1. 从 `index.html` 查看全部页面导航。
2. 进入 `pages/auth/login.html`，点击登录进入商户首页。
3. 从注册页进入验证码页，再进入店铺资料、资质上传、审核结果。
4. 在商户首页查看订单指标、快捷入口、最新订单和底部导航。
5. 进入订单列表、订单详情、分配厨师、服务进度，演示商户处理订单流程。
6. 查看厨师团队、套餐菜品、钱包结算、经营数据和我的页面。

## 说明

- 页面宽度按移动端 375px 优先设计，桌面浏览器中以手机壳居中显示。
- `assets/js/mock.js` 只负责本地静态交互反馈，例如筛选、上传卡片状态、厨师选择和 toast。
- 所有数据为静态演示数据，不代表真实经营数据。
