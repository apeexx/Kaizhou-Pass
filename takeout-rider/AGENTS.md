# takeout-rider 规则

本目录用于“外卖骑手端 / 配送端”手机端纯前端静态演示。

## 范围

- 只使用 HTML、CSS、原生 JavaScript。
- 不接后端、不接数据库、不接真实接口。
- 不做真实登录、鉴权、提现、文件上传、地图定位或导航接入。
- 页面数据来自静态 HTML 或 `assets/js/mock.js` 的本地演示交互。
- 页面跳转使用普通 `a` 标签 `href` 或静态表单跳转。
- 视觉主题继承仓库根目录 `AGENTS.md` 的“开州金厨设计规范”，默认使用轻电商 + 本地生活特卖风格、橙红主色、浅灰背景、白色圆角卡片和弱阴影。

## 页面定位

本目录面向外卖配送骑手，用于演示骑手登录注册、入驻审核、接单大厅、当前配送、取餐送达、路线查看、收入明细和个人中心流程。

不要把文案写成用户点餐端，也不要写成外卖商户端。后续修改需要保持外卖骑手端静态演示定位，不得新增后端、数据库、真实接口或真实地图能力。
